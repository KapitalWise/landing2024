Get in touch

Go to https://app.kapitalwise.com/admin/dashboard


1. Click on Campaigns on the left menue bar



2. Click on Create Campaign on the top tab



3. Name for your Campaign. Eg. "Looking for a Financial Advisor?". Please note that this name will appear on the final widget as the widget title.



4. Add a description for your reference.



5. Click on Design



6. Select the widget type.

You will need one Advisor Connect campaign as this will act as your lead funneling widget for other campaigns.



7. Select primary color

Select the primary color of your website. You can also copy paste the HTML code here.



8. Select Layout

You can pick Sidebar, Floating, or Embedded. The sidebar will dock the widget on the left sidebar, and Floating will generate a floating widget at the bottom right corner. Select Embedded if you plan to place the widget inline.



9. Select the Position

This will determine if the widget will be displayed on the left or right side of the page.



10. Click on Save to save the design



11. Click on Publish Content



12. Copy and paste the code into your content.

You can add the Javascript within your content or use the URL within social media or emails.
